Research a topic using available web tools.

## Instructions

1. Use the `mcp__fetch__fetch` tool to retrieve relevant web pages
2. Read and extract key information from the fetched content
3. Summarize findings in a clear, structured format
4. Cite sources with URLs

## Search strategy

- Start with the most authoritative sources for the topic
- Fetch 2-3 pages to cross-reference information
- If the first result is insufficient, try alternative URLs or search terms
- For current events, prefer news sites and official sources

## Output format

- Lead with a concise answer (1-2 sentences)
- Follow with supporting details in bullet points
- End with source URLs
